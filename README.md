# Part I: Get started with a simple concept
## I.1/ How a micromouse works
 - The role of a micromouse is to ***Navigate through a maze within a specified amount of time***.
 - Basic requirements for a microusemouse:
   1. Must autonomously detects walls (Using the sensors, could be an IR sensor or ultrasonic sensors)
      1.1 Avoids walls by movements such as turning 90, 180 or moving straight
   3. Detects whenever it has reached the goal (Algorithms)
   4. Maps out the maze (Optional)
